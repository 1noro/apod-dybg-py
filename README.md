# apod-dybg
Script to change daily the wallpaper of your Gnome Shell (Ubuntu) for the photo of the "Astronomy Picture of the Day" (https://apod.nasa.gov) or the "Amateur Astronomy Picture of the Day" (http://www.aapodx2.com)
